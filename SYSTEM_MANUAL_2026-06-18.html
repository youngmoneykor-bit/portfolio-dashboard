🧬 시스템 2 · BioLog GitHub 자동저장 실시간

1
PAT (Personal Access Token) 발급
GitHub → Settings → Developer settings → Fine-grained tokens
권한: Contents (Read & Write) · 대상 repo: portfolio-dashboard
토큰은 biolog.html 내 localStorage에만 저장 (서버 전송 없음)
2
데이터 입력 → 자동 커밋
+ 저장 버튼 클릭 시 GitHub REST API PUT 호출.
// UTF-8 안전 base64 인코딩 (한글 포함)
const encoded = btoa(unescape(encodeURIComponent(json)));

await fetch(`https://api.github.com/repos/${GH_OWNER}/${GH_REPO}/contents/${GH_PATH}`, {
  method: "PUT",
  headers: { Authorization: `Bearer ${token}` },
  body: JSON.stringify({
    message: `feat: biolog update ${date}`,
    content: encoded,
    sha: fileInfo.sha   // 기존 파일 SHA 필수
  })
});
3
페이지 열 때 자동 불러오기
GitHub에서 최신 biolog_data.json 자동 GET → 로컬 데이터와 날짜 기준 병합.
const decoded = decodeURIComponent(escape(atob(fileInfo.content.replace(/\n/g,""))));
const parsed  = JSON.parse(decoded);
